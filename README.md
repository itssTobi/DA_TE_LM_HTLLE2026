# Diplomarbeit_2026: Virtuelle Cybersecurity-Testumgebung mit IDS

Dieses Repository enthält die Projektdateien zur Diplomarbeit „Virtuelle Cybersecurity-Testumgebung mit IDS“.
Ziel des Projekts ist die Entwicklung einer sicheren, virtuellen Netzwerkumgebung, die im Unterricht eingesetzt werden kann und durch ein Intrusion Detection System (IDS) überwacht wird.

Inhalte

Blue Team – Aufbau und Konfiguration der sicheren Netzwerkumgebung inklusive IDS

Red Team – Durchführung von Angriffssimulationen

Dokumentation – Projektbeschreibung und Ergebnisse der Tests
